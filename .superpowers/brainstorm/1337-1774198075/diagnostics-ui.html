Backend Instrumentation

New events and data pushed via SignalR in real-time. Same API consumable by Radio.Web Blazor UI later.

New SignalR Events
  • SipMessageReceived (method, from, status, raw)
  • SipMessageSent (method, to, body summary)
  • SipDiagnosis (issue description, suggestions)
  • Ht801HealthUpdate (registration, ping, hook)
  • RtpStreamStats (pkt/s, jitter, loss, levels)
  • CallStateTimeline (timestamped event log)
  • GvBridgeAudioStats (ws frames/s, queue depth)
New API Endpoints
  • GET /api/diagnostics/status (full snapshot)
  • GET /api/diagnostics/sip-log (recent messages)
  • POST /api/diagnostics/test-ring (send test INVITE)
  • POST /api/diagnostics/test-audio (send RTP test tone)
  • GET /api/diagnostics/ht801 (read device config)
  • POST /api/diagnostics/ht801/validate (check+fix)
