Key New Component: GVAudioBridge

Inbound (Caller → Phone)
  1. Dequeue from InboundAudioQueue
  2. Decode base64 → raw PCM (16kHz 16-bit)
  3. Resample 16kHz → 8kHz
  4. Encode PCM → G.711 µ-law
  5. Send via RTP to HT801
Outbound (Phone → Caller)
  1. Receive RTP from HT801
  2. Decode G.711 µ-law → PCM (8kHz)
  3. Resample 8kHz → 16kHz
  4. Encode raw PCM → base64
  5. Send audioFrame via WebSocket
